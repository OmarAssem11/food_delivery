# Food Delivery app using Clean Architecture(TDD)

## Features:

* Register
* Login
* Logout
* Forgot password
* View all restaurants
* View restaurant details and products
* View product details
* Add product to cart
* View your cart and proceed to checkout
* View your orders and order details
* Track your order
* View profile
* Edit profile
* Localization (en, ar)
